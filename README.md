# Чарушников Никита Бас-2 Лабораторная №8

# Задание 1

## Задача 1

### Текст задачи

Разработать консольное приложение с дружественным интерфейсом с возможностью выбора
заданий для работы с «базой данных (БД)», хранящейся в бинарном файле. Перечень полей
(минимум 5), достаточно полно характеризующих заданную в варианте предметную область,
предложить самостоятельно (постараться отразить в перечне полей такие, которые требуют разных
типов данных). Приложение должно выполнять следующие функции:
1. Чтение базы данных из бинарного файла.
2. Просмотр базы данных.
3. Удаление элементов (по ключу).
4. Добавление элементов.
5. Реализация 4 запросов (формулировки запросов придумать самостоятельно). 2 запроса
должны возвращать перечень, 2 запроса одно значение.
В классе должны присутствовать свойства, конструкторы, перегруженный метод ToString(). Весь
функционал приложения реализовать в виде методов вспомогательного класса с помощью LINQзапросов.
Предусмотреть обработку возможных ошибок при работе программы.
Пункт 5 оценивается в 2 балла, остальные пункты по 1 баллу. Максимально за лабораторную работу
можно получить 6 баллов.

### Алгоритм решения

Класс DataBase (данные студента):
Создаётся класс DataBase с пятью приватными полями: _id, _lastName, _firstName, _avg, _grade
Для каждого поля реализуется свойство с get и set
Реализуется конструктор с пятью параметрами для инициализации полей через свойства
Переопределяется метод ToString который возвращает строку со всеми данными студента

Класс Cheker (вспомогательный класс):
Реализуется метод WriteToFile который принимает список студентов List<DataBase>, открывает/создаёт файл student.dat через BinaryWriter и записывает каждого студента поле за полем
Реализуется метод ReadFromFile который создаёт пустой список, открывает файл через BinaryReader, читает данные пока не достигнет конца файла, на каждом шаге создаёт объект DataBase и добавляет его в список, возвращает заполненный список
Реализуется метод ShowAll который проходит по списку и выводит каждого студента через Console.WriteLine
Реализуется метод Delete который находит студента по Id через list.Find, если не найден выводит сообщение, если найден удаляет из списка
Реализуется метод Add который считывает данные с консоли, создаёт новый объект DataBase и добавляет его в список
Реализуются четыре LINQ-запроса: студенты с баллом выше 7, студенты определённого класса, максимальный средний балл, количество студентов в классе

Метод Main:
Создаётся начальный список из пяти студентов типа List<DataBase>
Список записывается в файл через Cheker.WriteToFile
Запускается цикл с меню где пользователь выбирает действие: просмотр, удаление, добавление, LINQ-запросы или выход
После каждого изменения (удаление, добавление) список сохраняется в файл через Cheker.WriteToFile
